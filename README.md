# Photoshop Banana

- Photoshop jsx 格式插件
- 用gemini nano banana api 在photoshop中生成图片

## 安装

将PS_Banana.jsx复制到ps的scripts文件夹下，例如：
C:\Program Files\Adobe\Adobe Photoshop 2023\Presets\Scripts\

## 使用
- File>Scripts>PS_Banana  运行插件
- 可以给这个插件设置快捷键，以便快速使用。

## 功能
- 支持图层模式， 使用 source  / reference  命名图层组， 可以得到类似chat中发送两张图的效果
- 支持选区
- Prompt预设
- 自定义API提供商
